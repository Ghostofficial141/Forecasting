Supported Time-Series Models

Four enterprise architectures engineered to predict complex seasonal patterns.

SARIMA Statistical

Seasonal AutoRegressive Integrated Moving Average model. Best suited for high-density, pure seasonal sales patterns with strong stationary historical components.

Facebook Prophet Additive Model

Optimized for weekly, monthly, and yearly seasonal cycles. Built-in US federal holiday effects to robustly handle spike variations in sales trends.

XGBoost Regressor Machine Learning

Recursive multi-step forecast using optimized lag features, standard rolling statistics, and time components to capture sudden non-linear market movements.

LSTM Network Deep Learning

Long Short-Term Memory recurrent neural network designed for sequential lookback dependency, utilizing a deep stack architecture with dropout regularization.
